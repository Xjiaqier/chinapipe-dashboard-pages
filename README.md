# Chinapipe Dashboard Pages

This repository hosts the static dashboard generated by the local
`chinapipe_scraper` pipeline. The scraper keeps its browser profile and API
credentials locally. Its daily deployment task copies the generated
`panel.html` into `site/index.html`, commits the change, and pushes it here.

The GitHub Actions workflow deploys `site/` to GitHub Pages after each changed
dashboard is pushed to `main`.
